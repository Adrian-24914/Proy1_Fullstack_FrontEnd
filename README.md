# Series Tracker - Frontend

Cliente web Cyberpunk 2077 para gestión de series.

Deploy: https://proy1-fullstack-front-end.vercel.app
Backend: https://empowering-prosperity-production-d1ee.up.railway.app
Repositorio Backend: https://github.com/Adrian-24914/Proy1_Fullstack_BackEnd

Tecnologías:
- HTML5
- CSS3
- JavaScript Vanilla (sin librerías)
- Fetch API

Funcionalidades:
- Listar series
- Crear serie
- Editar serie
- Eliminar serie
- Buscar series
- Filtrar por género
- Exportar CSV
- Subir imágenes

CORS

CORS (Cross-Origin Resource Sharing) permite que el navegador acepte peticiones desde un origen distinto; el backend está configurado para permitir peticiones desde este dominio mediante los siguientes headers:

Access-Control-Allow-Origin: https://proy1-fullstack-front-end.vercel.app
Access-Control-Allow-Methods: GET, POST, PUT, DELETE, OPTIONS
Access-Control-Allow-Headers: Content-Type
